# Visual-Time-Machine

Visual-Time-Machine is a simple web-based project that demonstrates the concept of **Undo and Redo operations** using the **Stack Data Structure**.

This project helps students understand how stacks work in real applications such as text editors, history systems, and time-based state management.

## Features

- Push data into Stack A
- Undo operation (moves element from Stack A to Stack B)
- Redo operation (moves element from Stack B to Stack A)
- Clear stacks
- Visual stack representation
- Modern web interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. User enters a value and clicks **Push**.
2. The value is added to **Stack A**.
3. When **Undo** is clicked, the top element of Stack A moves to Stack B.
4. When **Redo** is clicked, the element from Stack B returns to Stack A.

This simulates a **time-machine like behavior**, allowing users to move backward and forward between states.
